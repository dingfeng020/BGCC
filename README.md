# 简介
BGCC（百度通用通信组件）是一个跨语言、跨平台、面向对象的高性能轻量级RPC框架。

>1. 具有自定义的接口描述语言和代码生成引擎
>2. 支持事务级别的服务端消息主动推送
>3. 消息流监控
>4. 变态的效率

# 使用产品
